Animaciones

Kazui ofrece una amplia variedad de animaciones css fáciles de usar.

@animations_examples.animations_example()

Uso

Las animaciones estan distribuidas en tres clases que son el tipo de animación,la función y la duración. Con esta combinación podemos crear asombrosas animaciones en segundos. 


<div class="ui-ease ui-fade-in ui-2s"></div>
