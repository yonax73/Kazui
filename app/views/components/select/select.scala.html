Uso

Para usar el Select solo tenemos que agregar una etiqueta HTML con un id y la clase ui-select 


<div id="select" class="ui-select"></div>

Javascript

Creamos un objeto Select, que recibe como parámetro el HTMLElement y un array de objetos.


var items = new Select(document.getElementById('select'), [
    {option: 1, value: 'item 1'},
    {option: 2, value: 'item 2'},
    {option: 3, value: 'item 3'},
    {option: 4, value: 'item 4',selected:true},
    {option: 5, value: 'item 5'}
]);

Métodos

Con el objeto ya creado podemos llamar cualquiera de sus métodos: items.getOption().

Nombre Descripción
addItem(option, value) Agrega un nuevo item.
focus() Enfoca el Select.
getItem() Retorna el item actual como un objeto. {option:currentOption,value:currentValue}
getOption() Retorna la opción del item actual.
getSize() Retorna el total de items que tiene el Select.
getValue() Retorna el valor del item actual.
isDisabled() Retorna verdadero si el Select esta inhabilitado
isOpen() Retorna verdadero si el Select esta abierto
isReadOnly() Retorna verdadero si el Select esta en solo lectura.
selectItem(option) Selecciona cualquier item por la opcioón.
setDisabled(disabled: boolean) Inhabilitado/habilita el Select
setHeight(height: string) Cambia la altura para ver los items, los valores permitidos son por ejemplo: '120px','9em','45%'.
setIcono(icono: string) Cambia el icono del Select. Kazui usa fontawesome. setIcono('fa-chevron-down').
setReadOnly(readOnly: boolean) Cambia el estado del Select a solo lectura.
toggle() Muestra/esconde los items.

Eventos

Nombre Descripción
onchange(handlerFunction) Este evento se dispara cada vez que hay un cambio en el Select, recibe como parametro una function(){}
