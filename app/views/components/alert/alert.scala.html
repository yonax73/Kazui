@alert_examples.alert_example()

Uso

El objeto Alert crea HTML dinámico para su propia construcción, solo es necesario agregar una etiqueta HTML con un id para referenciarlo en javascript.

	
<div id="alert"></div>

Javascript

Creamos un objeto Alert, que recibe como parámetro el HTMLElement

	
var objAlert = new Alert(document.getElementById('alert'));
*No Crear objeto con nombre alert por que dará confilcto con el alert() de javascript.

Métodos

Con el objeto ya creado podemos llamar cualquiera de sus métodos: objAlert.danger('message').

Nombre Descripción
success('message') Muestra una alerta de tipo Success.
info('message') Muestra una alerta de tipo Info.
warning('message') Muestra una alerta de tipo Warning.
danger('message') Muestra una alerta de tipo Danger.
wait('message') Muestra una alerta de tipo Wait.
close() Cierra la alerta.

Propiedades

Por medio de las propiedades podemos cambiar los iconos y las animaciones del objeto.

En los iconos

Kazui usa fontawesome.
Se puede acceder a las propiedades y cambiar los iconos del objeto usando la clase correcta objAlert.icoSuccess = 'fa-smile-o'

Nombre Descripción
icoSuccess Icono de alerta tipo Success
icoInfo Icono de alerta tipo Info
icoWarning Icono de alerta tipo Warning
icoDanger Icono de alerta tipo Danger
icoWait Icono de alerta tipo Wait,usar Spinning Icons

Para las animaciones

Kazui cuenta con su propia libreria de animaciones.
También se puede acceder a las propiedades y cambiar los iconos del objeto usando la clase correcta objAlert.animationIn = 'ui-fade-in'

Nombre Descripción
typeAnimation Tipo de animación a usar: ui-ease,ui-ease-in,etc.
animationIn Animación cuando se muestra la alerta: ui-fade-in,ui-fade-in-up,etc.
animationOut Animación cuando se cierra la alerta: ui-fade-out,ui-fade-out-up,etc.
